MUST HAVE
As a user, I should be able to add a contact
As a user, I should be able to see a list of contacts, so that I can choose one
As a user, I should be able to find a contact
As a user, I should be able to view the details of a contact

SHOULD HAVE
As a user, I should be able to add an email address to a contact
As a user, I should be able to add notes to a contact
As a user, I should be able to delete a contact
As a user, I should be able to edit a contact

COULD HAVE
As a user, I should be able to send an email to a contact
As a user, I should be able to add a photo to contact
As a user, I should be able to sort contacts alphabetically, so that ...
As a user, I should be able to group my contacts
As a user, I should be able to search for a contact

WON'T DO
As a user, I should be able to share a contact
As a user, I should be able to export the contact list
As a user, I should be able to import a contact list
